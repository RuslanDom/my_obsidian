## Run()
## –°–æ–∑–¥–∞–Ω–∏–µ –ø—Ä–æ—Ü–µ—Å—Å–∞

–î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –Ω–æ–≤–æ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Ñ—É–Ω–∫—Ü–∏—é¬†`subprocess.run()`. –í–æ—Ç –ø—Ä–∏–º–µ—Ä –∑–∞–ø—É—Å–∫–∞ —Å–∏—Å—Ç–µ–º–Ω–æ–π –∫–æ–º–∞–Ω–¥—ã¬†`ls`:

–ó–∞–ø—É—Å—Ç–∏—Ç—å

## –ü–æ–ª—É—á–µ–Ω–∏–µ –≤—ã–≤–æ–¥–∞

–î–ª—è —Ç–æ–≥–æ —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –≤—ã–≤–æ–¥ –∫–æ–º–∞–Ω–¥—ã, —É–∫–∞–∂–∏—Ç–µ –ø–∞—Ä–∞–º–µ—Ç—Ä¬†`capture_output=True`:

**result.stdout**¬†–±—É–¥–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –≤—ã–≤–æ–¥ –∫–æ–º–∞–Ω–¥—ã –≤ –≤–∏–¥–µ –±–∞–π—Ç–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏. –ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–ª—É—á–∏—Ç—å —Ç–µ–∫—Å—Ç–æ–≤—É—é —Å—Ç—Ä–æ–∫—É, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –º–µ—Ç–æ–¥¬†`decode()`:

## quote()

```python
import os  
import shlex, subprocess  
from typing import List  
from flask import Flask, request  
  
app = Flask(__name__)  
  
  
@app.route("/ps", methods=["GET"])  
def ps() -> str:  
	# –ü–æ–ª—É—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ —Å–ª–æ–≤ –æ—Ç –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –∏ –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç –∏—Ö –≤ —Å–ø–∏—Å–æ–∫
    args: List[str] = request.args.getlist("arg")  
	# –ó–∞—â–∏—â–∞—è –æ—Ç –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω—ã—Ö —É—è–∑–≤–∏–º–æ—Å—Ç–µ–π –∏–Ω—ä–µ–∫—Ü–∏–∏ –æ–±–æ–ª–æ—á–∫–∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–Ω—É—é –≤–µ—Ä—Å–∏—é —Å—Ç—Ä–æ–∫–∏ s
	
    safe_command = shlex.quote(''.join(args))  
    # check_output() –∑–∞—Ö–≤–∞—Ç—ã–≤–∞–µ—Ç –≤—ã–≤–æ–¥ –∫–æ–º–∞–Ω–¥—ã –∏ –≤—ã–∑—ã–≤–∞–µ—Ç –∏—Å–∫–ª—é—á–µ–Ω–∏–µ, –µ—Å–ª–∏ –∫–æ–º–∞–Ω–¥–∞ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∞  
    result = str(subprocess.check_output(["ps", safe_command]))    
    return "<br>".join(result.split(r'\n'))  
  
if __name__ == "__main__":  
    app.run(debug=True)
```
